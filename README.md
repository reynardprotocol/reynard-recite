# 📂 背诵沙盘系统 (Recitation Sandbox) 核心档案

**项目名称**：背诵沙盘系统 / 叶老师的背书室
**当前版本状态**：v2.0 (模块化重构版)
**运行环境**：纯前端浏览器环境 + 本地 Vite 开发服务器

## 🛠️ 技术栈
* **构建工具**：Vite (Node.js 环境仅用于本地编译)
* **CSS 框架**：Tailwind CSS v3 (PostCSS 编译以极致瘦身)
* **脚本逻辑**：Vanilla JavaScript (原生 ES6 模块机制)
* **数据存储**：浏览器的 `localStorage` (纯本地化存储，无后端)

## 🧱 系统核心模块与功能

### 1. 场景与 UI 底层
* 拟真木地板与立体课桌 UI (`wood-floor`, `wood-desk`)。
* 动态生成的 SVG 防撞色头像系统（按姓名哈希值或导入顺序生成）。
* 基于总分的动态成就边框（>=5分银色，>=15分金色）。

### 2. 班级与学生管理
* 支持多班级创建与切换。
* 支持手动添加单个学生，或通过 `.txt` 文件批量导入。
* 等候室（未入座）与课桌（已入座）的拖拽（Drag & Drop）分配逻辑。
* 双击头像可切换“组长”皇冠标识。

### 3. 打卡矩阵系统 (Matrix)
* 支持全局列标（M1, M2...）与当前班级统一步伐任务。
* 支持为特定学生添加/删除“专属任务”。
* 四档打卡状态切换：空星(0) -> 半星(0.5) -> 满星(1) -> 绿色对勾(2)。
* 打满 2 分时，触发鼠标位置的 25 颗随机色彩纸屑粒子爆发特效。

### 4. 命运大转盘 (Roulette)
* 仅抽取当前“已入座”的学生。
* Canvas 绘制的多彩扇形转盘，带物理减速旋转动画（4秒）。
* 抽取结果定格后，中心爆发粒子，并自动弹出该幸运儿的打卡矩阵。

### 5. 数据中心与排行榜
* 支持查看个人积分榜与小组总分榜。
* 全局设置面板（列标、统一任务拦截删除保护，防止误删已有数据的任务）。
